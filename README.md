# Meme Sağlığım - Mobil Uygulama Projesi
**Samsun Üniversitesi Yazılım Mühendisliği**

Bu proje, meme sağlığı farkındalığını artırmak ve kullanıcıların sağlık takibini kolaylaştırmak amacıyla geliştirilmiş bir Full-Stack mobil uygulamadır.

## 🛠 Kullanılan Teknolojiler
* **Frontend:** React Native, Expo
* **Build Araçları:** Android Studio, NDK, CMake, Ninja
* **Backend:** Node.js, Express.js (Full-stack yapılandırması)

## 🚀 Kurulum ve Çalıştırma
Uygulamayı yerel ortamda çalıştırmak için:
1. Proje dizinine gidin: `cd C:\frontend`
2. Bağımlılıkları yükleyin: `npm install`
3. Uygulamayı başlatın: `npx expo start`
4. Emülatörde açmak için klavyeden `a` tuşuna basın.

## ✨ Öne Çıkan Özellikler
* Kullanıcı dostu pembe temalı arayüz
* Dinamik form yapıları ve veri girişi
* Expo Go üzerinden anlık test imkanı
